## About Me

I'm a passionate **AI & Data Science professional** with **3+ years of experience** at the intersection of **Artificial Intelligence and Healthcare**. I architect and build production-grade intelligent systems — from 100+ agent autonomous healthcare platforms to LLM-powered clinical documentation tools — that turn complex medical data into life-improving outcomes.

- Specializing in **AI for Healthcare** — building systems where accuracy isn't optional
- Creator of **RuralScribe OS** — a 100+ agent autonomous healthcare operating system designed for rural communities
- Building **LLM-powered agents** with LangChain, Groq, and OpenAI that actually work in production
- Writing about AI, Healthcare & Data Science on **[Medium](https://medium.com/@a.v.vivekvarma)** — sharing real lessons from the field
